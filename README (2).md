# ✈️ King Hussein International Airport
## Electromechanical Maintenance System — v2.0

A comprehensive web-based maintenance management system for airport electromechanical assets.

### 🔗 Live Demo
[View Demo](https://[your-username].github.io/[repo-name]/)

### 📦 Features
- **Dashboard** — Real-time overview of all assets and statistics
- **Asset Management** — Track 70+ assets across 6 groups and 14 types
- **Maintenance** — Preventive & corrective maintenance records
- **Inspections** — Quality & safety inspection forms with history
- **Reports** — Generate system reports
- **Scientific Calculator** — Built-in iOS-style calculator (drag & drop)
- **Bilingual** — Full Arabic/English support

### 🏗️ Asset Groups
| Group | Types | Assets |
|-------|-------|--------|
| ⚡ Power Systems | Generators, UPS | 10 |
| 💡 Airfield Lighting | CCR, Runway, Taxiway, Apron Lights | 20 |
| ⚙️ Mechanical | Pumps, HVAC | 10 |
| 🔒 Security Screening | X-Ray, Walk Through | 10 |
| 🧳 BHS – Baggage Handling | Conveyors, Gear Motors, Sensors | 15 |
| 🔌 Electrical Panels | Panels | 5 |

### 🚀 Deploy to GitHub Pages
1. Upload `index.html` to your repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Access at `https://[username].github.io/[repo]/`

### 💻 Tech Stack
- Pure HTML5 / CSS3 / JavaScript (no dependencies)
- iOS-style UI components
- Responsive design (mobile + desktop)
